## HTML Boshqa Ro'yxatlar

HTML tavsif ro'yxatlarini ham qo'llab-quvvatlaydi.

#### HTML Tavsif Ro'yxatlari
Tavsif ro'yxati - bu terminlar ro'yxati bo'lib, har bir termin uchun tavsif mavjud.

`<dl>` tegi tavsif ro'yxatini belgilaydi, `<dt>` tegi terminni (nomni) belgilaydi, va `<dd>` tegi har bir terminni ta'riflaydi:

**Misol:**
```html
<dl>
  <dt>Coffee</dt>
  <dd>- qora issiq ichimlik</dd>
  <dt>Milk</dt>
  <dd>- oq sovuq ichimlik</dd>
</dl>
```

#### Bobni Yig'ish
- HTML `<dl>` elementidan tavsif ro'yxatini belgilash uchun foydalaning.
- HTML `<dt>` elementidan tavsif terminini belgilash uchun foydalaning.
- HTML `<dd>` elementidan terminning tavsifini berish uchun foydalaning.

#### HTML Mashqlari
Mashqlar orqali o'zingizni sinab ko'ring:

**Mashq:**
Tavsif ro'yxatidagi har bir terminni tavsiflovchi HTML elementlarini ishlating.

```html
<dl>
  <dt>Coffee</dt>

  - qora issiq ichimlik

  <dt>Milk</dt>

  - oq sovuq ichimlik

</dl>
```

#### HTML Ro'yxat Teglari

| Teg  | Ta'rif  |
|------|---------|
| `<ul>` | Tartibsiz ro'yxatni belgilaydi |
| `<ol>` | Tartibli ro'yxatni belgilaydi |
| `<li>` | Ro'yxat elementini belgilaydi |
| `<dl>` | Tavsif ro'yxatini belgilaydi |
| `<dt>` | Tavsif ro'yxatidagi terminni belgilaydi |
| `<dd>` | Tavsif ro'yxatidagi terminning ta'rifini belgilaydi |
