### HTML Ro'yxatlar

HTML ro'yxatlari veb-dasturchilarga bir qator tegishli elementlarni ro'yxat shaklida guruhlash imkonini beradi.

#### Misol
Tartibsiz HTML ro'yxati:

- Element
- Element
- Element
- Element

Tartibli HTML ro'yxati:

1. Birinchi element
2. Ikkinchi element
3. Uchinchi element
4. To'rtinchi element

#### Tartibsiz HTML Ro'yxat
Tartibsiz ro'yxat `<ul>` tegi bilan boshlanadi. Har bir ro'yxat elementi `<li>` tegi bilan boshlanadi.

Ro'yxat elementlari odatda qora nuqtalar (bullets) bilan belgilangan bo'ladi:

**Misol:**
```html
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
```

#### Tartibli HTML Ro'yxat
Tartibli ro'yxat `<ol>` tegi bilan boshlanadi. Har bir ro'yxat elementi `<li>` tegi bilan boshlanadi.

Ro'yxat elementlari odatda raqamlar bilan belgilangan bo'ladi:

**Misol:**
```html
<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>
```

#### HTML Tavsif Ro'yxatlari
HTML shuningdek, tavsif ro'yxatlarini ham qo'llab-quvvatlaydi.

Tavsif ro'yxati terminlar ro'yxati bo'lib, har bir terminning ta'rifi bilan birga bo'ladi.

`<dl>` tegi tavsif ro'yxatini belgilaydi, `<dt>` tegi terminni (nomini) belgilaydi, va `<dd>` tegi har bir terminni tasvirlaydi:

**Misol:**
```html
<dl>
  <dt>Coffee</dt>
  <dd>- qora issiq ichimlik</dd>
  <dt>Milk</dt>
  <dd>- oq sovuq ichimlik</dd>
</dl>
```

#### HTML Mashqlar
Mashqlar orqali o'zingizni sinab ko'ring:

**Mashq:**
`<ul>` elementining ichiga "Coffee" matnli ro'yxat elementini qo'shing.

```html
<ul>
  <li>Coffee</li>
</ul>
```

#### HTML Ro'yxat Teglari

| Teg  | Ta'rif  |
|------|---------|
| `<ul>` | Tartibsiz ro'yxatni belgilaydi |
| `<ol>` | Tartibli ro'yxatni belgilaydi |
| `<li>` | Ro'yxat elementini belgilaydi |
| `<dl>` | Tavsif ro'yxatini belgilaydi |
| `<dt>` | Tavsif ro'yxatidagi terminni belgilaydi |
| `<dd>` | Tavsif ro'yxatidagi terminning ta'rifini belgilaydi |
