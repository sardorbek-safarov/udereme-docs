## HTML Kodlash (Belgilar To'plami)

HTML sahifasini to'g'ri ko'rsatish uchun veb-brauzer qaysi belgi to'plamidan foydalanishni bilishi kerak.

### HTML `charset` Atributi

Belgi to'plami `<meta>` tegi ichida ko'rsatiladi:

**Misol:**

```html
<meta charset="UTF-8" />
```

HTML5 spetsifikatsiyasi veb-ishlab chiquvchilarni UTF-8 belgi to'plamidan foydalanishga chaqiradi.

UTF-8 dunyodagi deyarli barcha belgilar va simbolarni qamrab oladi!

### Unicode Veb O'sishi

**To'liq UTF-8 Ma'lumotnomasi**

### ASCII Belgilar To'plami

ASCII veb uchun birinchi belgi kodlash standarti edi. U internetda foydalanish mumkin bo'lgan 128 ta turli belgi va harflarni aniqladi:

- Ingliz harflari (A-Z)
- Raqamlar (0-9)
- Maxsus belgilar, masalan, `! $ + - ( ) @ < >.`

### ANSI Belgilar To'plami

ANSI (Windows-1252) asl Windows belgi to'plami edi:

- Birinchi 127 belgi uchun ASCII bilan bir xil
- 128 dan 159 gacha bo'lgan maxsus belgilar
- 160 dan 255 gacha UTF-8 bilan bir xil

**Misol:**

```html
<meta charset="Windows-1252" />
```

### ISO-8859-1 Belgilar To'plami

ISO-8859-1 HTML 4 uchun sukut bo'yicha belgi to'plami edi. Bu belgi to'plami 256 ta turli belgi kodlarini qo'llab-quvvatladi. HTML 4 shuningdek UTF-8 ni ham qo'llab-quvvatladi.

- Birinchi 127 belgi uchun ASCII bilan bir xil
- 128 dan 159 gacha bo'lgan belgilar ishlatilmaydi
- 160 dan 255 gacha ANSI va UTF-8 bilan bir xil

**HTML 4 Misol:**

```html
<meta http-equiv="Content-Type" content="text/html;charset=ISO-8859-1" />
```

**HTML 5 Misol:**

```html
<meta charset="ISO-8859-1" />
```

### UTF-8 Belgilar To'plami

- 0 dan 127 gacha bo'lgan qiymatlar uchun ASCII bilan bir xil
- 128 dan 159 gacha bo'lgan belgilar ishlatilmaydi
- 160 dan 255 gacha ANSI va 8859-1 bilan bir xil
- 256 dan 10 000 gacha bo'lgan belgilarni davom ettiradi

**Misol:**

```html
<meta charset="UTF-8" />
```

**To'liq HTML Belgilar To'plami Ma'lumotnomasi.**
