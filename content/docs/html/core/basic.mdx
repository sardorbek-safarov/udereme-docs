## HTML Asosiy Misollar
Bu bobda biz ba'zi asosiy HTML misollarini ko'rsatamiz.

Biz hali o'rganmagan teglarimizdan foydalanadigan bo'lsak, xavotir olmang.

### HTML Hujjatlari
Barcha HTML hujjatlari hujjat turi deklaratsiyasi bilan boshlanishi kerak: `<!DOCTYPE html>`.

HTML hujjatining o'zi `<html>` bilan boshlanadi va `</html>` bilan tugaydi.

HTML hujjatining ko‘rinadigan qismi `<body>` va `</body>` orasida joylashgan.

Misol:
```html
<!DOCTYPE html>
<html>
<body>

<h1>Mening birinchi sarlavham</h1>
<p>Mening birinchi paragrafim.</p>

</body>
</html>
```

## `<!DOCTYPE>` Deklaratsiyasi
`<!DOCTYPE>` deklaratsiyasi hujjat turini ifodalaydi va brauzerlarga veb-sahifalarni to‘g‘ri ko‘rsatishda yordam beradi.

U faqat bir marta, sahifaning yuqori qismida (barcha HTML teglardan oldin) paydo bo'lishi kerak.

`<!DOCTYPE>` deklaratsiyasi kichik va katta harfga sezgir emas.

HTML5 uchun `<!DOCTYPE>` deklaratsiyasi quyidagicha:

`<!DOCTYPE html>`

## HTML Sarlavhalar
HTML sarlavhalar `<h1>` dan `<h6>` gacha bo'lgan teglar bilan belgilanadi.

`<h1>` eng muhim sarlavhani, `<h6>` esa eng kam muhim sarlavhani belgilaydi:

Misol:
```html
<h1>Bu 1-sarlavha</h1>
<h2>Bu 2-sarlavha</h2>
<h3>Bu 3-sarlavha</h3>
```

### HTML Paragraflar
HTML paragraflar `<p>` tegi bilan belgilanadi:

Misol:
```html
<p>Bu bir paragraf.</p>
<p>Bu yana bir paragraf.</p>
```

### HTML Havolalar
HTML havolalar `<a>` tegi bilan belgilanadi:

Misol:
```html
<a href="https://udemere.uz">Bu havola</a>
```

Havolaning manzili `href` atributida belgilanadi.

Atributlar HTML elementlari haqida qo'shimcha ma'lumot berish uchun ishlatiladi.

Siz atributlar haqida keyingi boblarda ko'proq bilib olasiz.

### HTML Rasmlar
HTML rasmlar `<img>` tegi bilan belgilanadi.

Manba fayl (`src`), alternativ matn (`alt`), kenglik (`width`), va balandlik (`height`) atributlar sifatida taqdim etiladi:

Misol:
```html
<img src="udemere.jpg" alt="udemere.uz" width="104" height="142">
```

### HTML Manbasini Ko‘rish
Hech qachon veb-sahifani ko'rib, "Bu qanday qilib amalga oshirilgan?" deb hayron bo'lganmisiz?

**HTML Manba Kodini Ko'rish:**
HTML sahifasida `CTRL + U` tugmalarini bosing yoki sahifani o'ng tugmasini bosib "View Page Source" ni tanlang. Bu sahifaning HTML manba kodini o'z ichiga olgan yangi oyna ochadi.

**HTML Elementni Tekshirish:**
Elementni (yoki bo‘sh joyni) o‘ng tugmasini bosib, "Inspect" ni tanlang, bu elementlarning nimadan tashkil topganini ko‘rish imkonini beradi (siz HTML va CSS ni ko‘rasiz). Shuningdek, ochilgan "Elements" yoki "Styles" panelida HTML yoki CSS ni to‘g‘ridan-to‘g‘ri tahrirlashingiz mumkin.
