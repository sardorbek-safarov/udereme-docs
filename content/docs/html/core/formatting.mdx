### HTML Matn Formatlash

HTML-da maxsus ma'noga ega bo'lgan matnni aniqlash uchun bir nechta elementlar mavjud.

#### Misol
```html
<b>Bu matn qalin</b>
<i>Bu matn kursiv</i>
<sub>Bu matn pastki indeks</sub> va <sup>bu matn yuqori indeks</sup>
```

### HTML Formatlash Elementlari

Formatlash elementlari maxsus turdagi matnlarni ko'rsatish uchun mo'ljallangan:

- `<b>` - Qalin matn
- `<strong>` - Muhim matn
- `<i>` - Kursiv matn
- `<em>` - Ta'kidlangan matn
- `<mark>` - Belgilangan matn
- `<small>` - Kichik matn
- `<del>` - O'chirilgan matn
- `<ins>` - Qo'shilgan matn
- `<sub>` - Pastki indeks matn
- `<sup>` - Yuqori indeks matn

### HTML `<b>` va `<strong>` Elementlari

HTML `<b>` elementi qo'shimcha ahamiyatsiz qalin matnni belgilaydi.

#### Misol
```html
<b>Bu matn qalin</b>
```

HTML `<strong>` elementi kuchli ahamiyatga ega matnni belgilaydi. Ichidagi kontent odatda qalin ko'rinishda bo'ladi.

#### Misol
```html
<strong>Bu matn muhim!</strong>
```

### HTML `<i>` va `<em>` Elementlari

HTML `<i>` elementi matnning bir qismini alternativ ovoz yoki kayfiyatda belgilaydi. Ichidagi kontent odatda kursiv ko'rinishda bo'ladi.

**Maslahat:** `<i>` tegidan texnik atama, boshqa tilga oid ibora, fikr, kema nomi va boshqalarni belgilash uchun foydalaniladi.

#### Misol
```html
<i>Bu matn kursiv</i>
```

HTML `<em>` elementi ta'kidlangan matnni belgilaydi. Ichidagi kontent odatda kursiv ko'rinishda bo'ladi.

**Maslahat:** Ekran o'quvchi `<em>` ichidagi so'zlarni urg'u bilan talaffuz qiladi.

#### Misol
```html
<em>Bu matn ta'kidlangan</em>
```

### HTML `<small>` Elementi

HTML `<small>` elementi kichikroq matnni belgilaydi:

#### Misol
```html
<small>Bu kichik matn.</small>
```

### HTML `<mark>` Elementi

HTML `<mark>` elementi belgilanadigan yoki ajratib ko'rsatiladigan matnni belgilaydi:

#### Misol
```html
<p>Bugun <mark>sut</mark> sotib olishni unutmang.</p>
```

### HTML `<del>` Elementi

HTML `<del>` elementi hujjatdan o'chirilgan matnni belgilaydi. Brauzerlar odatda o'chirilgan matn ustidan chiziq chizadi:

#### Misol
```html
<p>Mening sevimli rangim <del>ko'k</del> qizil.</p>
```

### HTML `<ins>` Elementi

HTML `<ins>` elementi hujjatga qo'shilgan matnni belgilaydi. Brauzerlar odatda qo'shilgan matnni tagiga chizadi:

#### Misol
```html
<p>Mening sevimli rangim <del>ko'k</del> <ins>qizil</ins>.</p>
```

### HTML `<sub>` Elementi

HTML `<sub>` elementi pastki indeks matnni belgilaydi. Pastki indeks matn odatdagi chiziqdan yarim belgi pastda paydo bo'ladi va ba'zida kichikroq shrift bilan ko'rsatiladi. Pastki indeks matn kimyoviy formulalar, masalan, H<sub>2</sub>O kabi ishlatilishi mumkin:

#### Misol
```html
<p>Bu <sub>pastki indeks</sub> matn.</p>
```

### HTML `<sup>` Elementi

HTML `<sup>` elementi yuqori indeks matnni belgilaydi. Yuqori indeks matn odatdagi chiziqdan yarim belgi yuqorida paydo bo'ladi va ba'zida kichikroq shrift bilan ko'rsatiladi. Yuqori indeks matn izohlar uchun, masalan, WWW<sup>[1]</sup> kabi ishlatilishi mumkin:

#### Misol
```html
<p>Bu <sup>yuqori indeks</sup> matn.</p>
```

### HTML Mashqlar

Mashq: Quyidagi parcha matnda "degradatsiya" so'zini qo'shimcha ahamiyat bilan belgilang.

```html
<p>
WWFning maqsadi bizning sayyoramizning tabiiy muhitini degradatsiyadan to'xtatishdir.
</p>
```
