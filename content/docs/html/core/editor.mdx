## HTML Muharrirlari

Oddiy matn muharriri HTMLni o'rganish uchun yetarli.

**HTMLni Notepad yoki TextEdit yordamida o'rganing**
Veb-sahifalarni professional HTML muharrirlari yordamida yaratish va o'zgartirish mumkin.

Biroq, HTMLni o'rganish uchun biz oddiy matn muharriridan, masalan, Notepad (PC) yoki TextEdit (Mac) tavsiya qilamiz.

Bizningcha, oddiy matn muharriri yordamida HTMLni o'rganish yaxshi usul.

Quyidagi bosqichlarni bajaring va Notepad yoki TextEdit yordamida o'zining birinchi veb-sahifangizni yarating.

**1-bosqich: Notepad-ni oching (PC)**
Windows 8 yoki undan keyin:

- Start ekranini oching (ekraningizning pastki chap burchagidagi oyna belgisi). "Notepad" deb yozing.

Windows 7 yoki undan oldin:

- Start > Dasturlar > Aksessuarlar > Notepad-ni oching.

**1-bosqich: TextEdit-ni oching (Mac)**

- Finder > Ilovalar > TextEdit-ni oching.

Shuningdek, fayllarni to'g'ri saqlash uchun ba'zi sozlamalarni o'zgartiring. "Preferences > Format" da "Plain Text" ni tanlang.

Keyin "Open and Save" da "HTML fayllarini formatlangan matn o'rniga HTML kodi sifatida ko'rsatish" katakchasini belgilang.

Keyin kodni joylashtirish uchun yangi hujjat oching.

**2-bosqich: Bir oz HTML yozing**
Quyidagi HTML kodini Notepad-ga yozing yoki nusxa ko'chiring:

```html
<!doctype html>
<html>
  <body>
    <h1>Mening birinchi sarlavham</h1>

    <p>Mening birinchi paragrafim.</p>
  </body>
</html>
```

**3-bosqich: HTML sahifani saqlang**
Faylni kompyuteringizga saqlang. Notepad menyusida "Fayl > Saqlash" ni tanlang.

Fayl nomini "index.htm" deb nomlang va kodlashni UTF-8 ga o'rnating (bu HTML fayllari uchun tavsiya etilgan kodlash).

**Maslahat:** Fayl kengaytmasi sifatida `.htm` yoki `.html` dan foydalanishingiz mumkin. Farqi yo'q; bu sizning tanlovingizga bog'liq.

**4-bosqich: HTML sahifani brauzeringizda ko'ring**
Saqlangan HTML faylini sevimli brauzeringizda oching (faylni ikki marta bosing yoki o'ng tugmasini bosib - "Open with" ni tanlang).

## VSCode Muharriri - "O'zingiz sinab ko'ring"

Bu kodni tezda sinab ko'rmoqchi bo'lganingizda mukammal vositadir. U shuningdek, ranglarni kodlash va kodni saqlash va boshqalar bilan bo'lishish imkoniyatiga ega:

Misol:

```html
<!doctype html>
<html>
  <head>
    <title>Sahifa sarlavhasi</title>
  </head>
  <body>
    <h1>Bu Sarlavha</h1>
    <p>Bu Paragraf.</p>
  </body>
</html>
```
