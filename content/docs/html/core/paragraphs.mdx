### HTML Paragraflar
Paragraf har doim yangi satrda boshlanadi va odatda matn blokidan iborat bo'ladi.

HTMLda `<p>` elementi paragrafni aniqlaydi.

Paragraf har doim yangi satrda boshlanadi va brauzerlar avtomatik ravishda paragraf oldida va keyin bir oz bo'sh joy (chekka) qo'shadi.

**Misol:**

```html
<p>Bu paragraf.</p>
<p>Bu esa boshqa paragraf.</p>
```

### HTML Ko'rsatilishi
HTML qanday ko'rsatilishini oldindan bilib bo'lmaydi.

Katta yoki kichik ekranlar va qayta o'lchamlangan oynalar turli xil natijalar yaratadi.

HTML bilan siz qo'shimcha bo'sh joylar yoki qo'shimcha satrlar qo'shish orqali ko'rsatishni o'zgartira olmaysiz.

Brauzer sahifa ko'rsatilganda har qanday qo'shimcha bo'sh joy va satrlarni avtomatik ravishda olib tashlaydi:

**Misol:**

```html
<p>
Bu paragraf
manba kodida ko'p satrlarni o'z ichiga oladi,
lekin brauzer
bunga e'tibor bermaydi.
</p>

<p>
Bu paragraf
manba kodida ko'p bo'shliqlarni o'z ichiga oladi,
lekin brauzer
bunga e'tibor bermaydi.
</p>
```

### HTML Gorizontal Chiziqlar
`<hr>` tegi HTML sahifasida tematik uzilishni aniqlaydi va ko'pincha gorizontal chiziq sifatida ko'rsatiladi.

`<hr>` elementi HTML sahifasidagi mazmunni ajratish yoki o'zgarishni aniqlash uchun ishlatiladi:

**Misol:**

```html
<h1>Bu sarlavha 1</h1>
<p>Bu bir oz matn.</p>
<hr>
<h2>Bu sarlavha 2</h2>
<p>Bu esa boshqa matn.</p>
<hr>
```

`<hr>` tegi bo'sh tegdir, ya'ni uning yopilish tegi yo'q.

### HTML Satr Bo'shliqlari
HTMLda `<br>` elementi satr bo'shlig'ini aniqlaydi.

Agar yangi paragrafni boshlamasdan satr bo'shlig'i (yangi satr) qo'shmoqchi bo'lsangiz, `<br>` dan foydalaning:

**Misol:**

```html
<p>Bu <br> satr bo'shliqlari bilan <br> paragraf.</p>
```

`<br>` tegi bo'sh tegdir, ya'ni uning yopilish tegi yo'q.

### She'r Muammosi
Ushbu she'r bitta satrda ko'rsatiladi:

**Misol:**

```html
<p>
  Mening Bonniyim okeanning ortida yotadi.

  Mening Bonniyim dengizning ustida yotadi.

  Mening Bonniyim okeanning ortida yotadi.

  Oh, Bonniyimni menga qaytar.
</p>
```

### Yechim - HTML `<pre>` Elementi
HTMLda `<pre>` elementi oldindan formatlangan matnni aniqlaydi.

`<pre>` elementidagi matn o'zgarmas kenglikdagi shrift bilan ko'rsatiladi (odatda Courier) va u bo'shliqlarni va satr bo'shliqlarini saqlab qoladi:

**Misol:**

```html
<pre>
  Mening Bonniyim okeanning ortida yotadi.

  Mening Bonniyim dengizning ustida yotadi.

  Mening Bonniyim okeanning ortida yotadi.

  Oh, Bonniyimni menga qaytar.
</pre>
```

### HTML Mashqlari
O'zingizni mashq qilib ko'ring.

**Mashq:**

To'g'ri HTML tegidan foydalanib, "Hello World!" matni bilan paragraf qo'shing.

```html
<html>
<body>
  <p>Hello World!</p>
</body>
</html>
```

### HTML Teglari Ma'lumotnomasi
Udemere'ning teglari ma'lumotnomasi HTML elementlari va ularning atributlari haqida qo'shimcha ma'lumotlarni o'z ichiga oladi.

| Teg  |	Tavsif |
|------|-------------|
| `<p>`	| Paragrafni aniqlaydi |
| `<hr>`	| Kontentdagi tematik o'zgarishni aniqlaydi |
| `<br>`	| Bitta satr bo'shligini qo'shadi |
| `<pre>`	| Oldindan formatlangan matnni aniqlaydi |
