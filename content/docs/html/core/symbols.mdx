## HTML Belgilari
Klaviaturangizda mavjud bo'lmagan belgilar yoki harflarni HTML-da entity-lar yordamida qo'shish mumkin.

### HTML Belgilari Entity-lari
HTML entity-lari oldingi bo'limda ko'rib chiqilgan.

Ko'pgina matematik, texnik va valyuta belgilar klaviaturada mavjud emas.

Bunday belgilarni HTML sahifasiga qo'shish uchun siz belgi uchun entity nomi yoki entity raqamini (o'nlik yoki o'n oltilik ma'lumotnoma) ishlatishingiz mumkin:

### Misol
Yevro belgisi (euro)ni ko'rsatish:

```html
<p>Men yevroni ko'rsataman: &euro;</p>
<p>Men yevroni ko'rsataman: &#8364;</p>
<p>Men yevroni ko'rsataman: &#x20AC;</p>
```

Bu quyidagicha ko'rinadi:
```
Men yevroni ko'rsataman: €
Men yevroni ko'rsataman: €
Men yevroni ko'rsataman: €
```

| Belgilar  | Raqami  | Entity       | Tavsif                   |
|-----------|---------|--------------|--------------------------|
| ∀         | &#8704; | &forall;     | Barchasi uchun           |
| ∂         | &#8706; | &part;       | Qisman differensial      |
| ∃         | &#8707; | &exist;      | Mavjud                   |
| ∅         | &#8709; | &empty;      | Bo'sh to'plamlar          |
| ∇         | &#8711; | &nabla;      | Nabla                    |
| ∈         | &#8712; | &isin;       | Elementi                 |
| ∉         | &#8713; | &notin;      | Element emas             |
| ∋         | &#8715; | &ni;         | A'zosi sifatida o'z ichiga oladi |
| ∏         | &#8719; | &prod;       | Ko'paytma                |
| ∑         | &#8721; | &sum;        | Yig'indi                 |
