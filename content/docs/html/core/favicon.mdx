## HTML Favicon

Favicon – bu brauzerning yorlig'ida sahifa sarlavhasi yonida ko'rsatiladigan kichik rasm.

<Image src='/images/html/favicon.png' width="772" height="156" alt="Favicon" />

### HTML Faviconni Qanday Qo'shish Mumkin

Siz favicon sifatida istalgan rasmni ishlatishingiz mumkin. Shuningdek, o'zingizning favicon rasmingizni https://www.favicon.cc kabi saytlarda yaratishingiz mumkin.

<Callout title='Maslahat'>
  Favicon kichik rasm bo'lgani uchun oddiy, yuqori kontrastli rasm bo'lishi kerak.
</Callout>

Favicon rasmi brauzer yorlig'ida sahifa sarlavhasi chap tomonida ko'rsatiladi, masalan:

**Favicon misoli**
Veb-saytingizga favicon qo'shish uchun favicon rasmni veb-serveringizning ildiz katalogiga saqlang yoki ildiz katalogida `images` nomli papka yarating va favicon rasmni ushbu papkaga saqlang. Favicon rasmi uchun keng tarqalgan nom "favicon.ico" hisoblanadi.

Keyin, `index.html` faylingizdagi `<title>` elementidan keyin `<link>` elementini qo'shing:

**Misol**

```html
<!DOCTYPE html>
<html>
<head>
  <title>Mening Sahifa Sarlavham</title>
  <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
</head>
<body>

<h1>Bu Sarlavha</h1>
<p>Bu Paragraf.</p>

</body>
</html>
```

Endi, `index.html` faylini saqlang va brauzeringizda yangilang. Brauzer yorlig'ida favicon rasm sahifa sarlavhasi chap tomonida ko'rsatiladi.

## Bo'lim Xulosasi

**HTML Mashqlari**

**Mashq:**
Qaysi HTML tegi faviconni (brauzer yorlig'ining chap tomonidagi kichik rasmni) boshqarish uchun ishlatiladi?

```html
<link rel="icon" type="image/x-icon" href="/images/favicon.ico">
```
