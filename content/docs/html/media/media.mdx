## HTML Multimedia

**Multimedia** vebda ovoz, musiqa, videolar, filmlar va animatsiyalar kabi elementlarni ifodalaydi.

### Multimedia nima?

Multimedia ko'plab turli formatlarda mavjud. U siz eshitishingiz yoki ko'rishingiz mumkin bo'lgan deyarli hamma narsani, masalan, rasm, musiqa, ovoz, video, rekordlar, filmlar, animatsiyalar va boshqalarni o'z ichiga oladi.

Veb sahifalar ko'pincha turli turdagi va formatdagi multimedia elementlarini o'z ichiga oladi.

### Brauzer Qo'llab-Quvvatlash

Birinchi veb-brauzerlar faqat matnni qo'llab-quvvatladi va bitta rangdagi bitta shrift bilan cheklangan edi.

Keyinchalik ranglar, shriftlar, rasmlar va multimedia qo'llab-quvvatlaydigan brauzerlar paydo bo'ldi!

### Multimedia Formatlari

Multimedia elementlari (masalan, audio yoki video) media fayllarida saqlanadi.

Fayl turini aniqlashning eng oddiy usuli fayl kengaytmasiga qarashdir.

Multimedia fayllari quyidagi format va kengaytmalarga ega: .wav, .mp3, .mp4, .mpg, .wmv, va .avi.

### Ommaviy Video Formatlari

Ko'plab video formatlari mavjud.

MP4, WebM va Ogg formatlari HTML tomonidan qo'llab-quvvatlanadi.

MP4 formatini YouTube tavsiya qiladi.

| Format          | Fayl        | Tavsif                                                                                                                                                                                                       |
| --------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| MPEG            | .mpg, .mpeg | MPEG. Moving Pictures Expert Group tomonidan ishlab chiqilgan. Vebda birinchi mashhur video formati. HTML-da qo'llab-quvvatlanmaydi.                                                                         |
| AVI             | .avi        | AVI (Audio Video Interleave). Microsoft tomonidan ishlab chiqilgan. Video kameralar va TV apparatlarida keng qo'llaniladi. Windows kompyuterlarida yaxshi o'ynaydi, lekin brauzerlarda emas.                 |
| WMV             | .wmv        | WMV (Windows Media Video). Microsoft tomonidan ishlab chiqilgan. Windows kompyuterlarida yaxshi o'ynaydi, lekin brauzerlarda emas.                                                                           |
| QuickTime       | .mov        | QuickTime. Apple tomonidan ishlab chiqilgan. Apple kompyuterlarida yaxshi o'ynaydi, lekin brauzerlarda emas.                                                                                                 |
| RealVideo       | .rm, .ram   | RealVideo. Real Media tomonidan video oqimini kam kengaytmali tarmoqlarda qo'llash uchun ishlab chiqilgan. Brauzerlarda o'ynamaydi.                                                                          |
| Flash           | .swf, .flv  | Flash. Macromedia tomonidan ishlab chiqilgan. Brauzerlarda o'ynash uchun qo'shimcha komponent (plagin) talab qilinishi mumkin.                                                                               |
| Ogg             | .ogg        | Theora Ogg. Xiph.Org Foundation tomonidan ishlab chiqilgan. HTML tomonidan qo'llab-quvvatlanadi.                                                                                                             |
| WebM            | .webm       | WebM. Mozilla, Opera, Adobe va Google tomonidan ishlab chiqilgan. HTML tomonidan qo'llab-quvvatlanadi.                                                                                                       |
| MPEG-4 yoki MP4 | .mp4        | MP4. Moving Pictures Expert Group tomonidan ishlab chiqilgan. Video kameralar va TV apparatlarida keng qo'llaniladi. Barcha brauzerlar tomonidan qo'llab-quvvatlanadi va YouTube tomonidan tavsiya qilinadi. |

<Callout type="warn" title="Eslatma">
  Faqat MP4, WebM va Ogg video formatlari HTML standarti tomonidan qo'llab-quvvatlanadi.
</Callout>

### Ommaviy Audio Formatlari

MP3 eng yaxshi format kompressiyalangan yozilgan musiqa uchun hisoblanadi. MP3 atamasi raqamli musiqa bilan sinonim bo'lib qoldi.

Agar sizning veb-saytingiz yozilgan musiqa haqida bo'lsa, MP3 eng yaxshi tanlovdir.

| Format    | Fayl        | Tavsif                                                                                                                                                                                                                                              |
| --------- | ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| MIDI      | .mid, .midi | MIDI (Musical Instrument Digital Interface). Elektron musiqa qurilmalari uchun asosiy format. MIDI fayllar ovozni o'z ichiga olmaydi, balki elektron qurilmalar tomonidan o'ynaladigan raqamli notalarni o'z ichiga oladi. Brauzerlarda o'ynamaydi. |
| RealAudio | .rm, .ram   | RealAudio. Real Media tomonidan kam kengaytmali tarmoqlarda audio oqimini qo'llash uchun ishlab chiqilgan. Brauzerlarda o'ynamaydi.                                                                                                                 |
| WMA       | .wma        | WMA (Windows Media Audio). Microsoft tomonidan ishlab chiqilgan. Windows kompyuterlarida yaxshi o'ynaydi, lekin brauzerlarda emas.                                                                                                                  |
| AAC       | .aac        | AAC (Advanced Audio Coding). Apple tomonidan iTunes uchun standart format sifatida ishlab chiqilgan. Apple kompyuterlarida yaxshi o'ynaydi, lekin brauzerlarda emas.                                                                                |
| WAV       | .wav        | WAV. IBM va Microsoft tomonidan ishlab chiqilgan. Windows, Macintosh va Linux operatsion tizimlarida yaxshi o'ynaydi. HTML tomonidan qo'llab-quvvatlanadi.                                                                                          |
| Ogg       | .ogg        | Ogg. Xiph.Org Foundation tomonidan ishlab chiqilgan. HTML tomonidan qo'llab-quvvatlanadi.                                                                                                                                                           |
| MP3       | .mp3        | MP3 fayllar asosan MPEG fayllarining ovoz qismidir. MP3 musiqa pleyerlarida eng mashhur formatdir. Yaxshi kompressiyani (kichik fayllar) yuqori sifat bilan birlashtiradi. Barcha brauzerlar tomonidan qo'llab-quvvatlanadi.                        |
| MP4       | .mp4        | MP4 video format bo'lsa ham, audio uchun ham ishlatilishi mumkin. Barcha brauzerlar tomonidan qo'llab-quvvatlanadi.                                                                                                                                 |

<Callout type="warn" title="Eslatma">
  Faqat MP3, WAV, va Ogg audio formatlari HTML standarti tomonidan qo'llab-quvvatlanadi.
</Callout>
