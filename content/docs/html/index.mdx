HTML (HyperText Markup Language) bu veb-sahifalarni yaratish uchun asosiy markup tili hisoblanadi. HTML yordamida veb-sahifaning tuzilishi, ya'ni matn, rasm, video, tugmalar va boshqa elementlar qanday ko'rinishini belgilash mumkin. HTML fayllarida turli xil teglardan foydalanib, sahifa elementlarini belgilash va ularni tartibga solish mumkin.

Masalan, `<h1>` tegidan sarlavha uchun, `<p>` tegidan paragraflar uchun, `<a>` tegidan esa havola (link) yaratish uchun foydalaniladi.

HTML dasturlash tili emas, balki markup tili bo'lib, u sahifaning tuzilishini ko'rsatadi va turli xil brauzerlar tomonidan sahifani qanday ko'rsatish kerakligini belgilaydi.

```html
<!doctype html>
<html>
  <head>
    <title>Sahifa sarlavhasi</title>
  </head>
  <body>
    <h1>Bu 1-darajali sarlavha tegi.</h1>
    <p>Bu paragraf tegi.</p>
  </body>
</html>
```
